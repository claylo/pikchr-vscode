# pikchr-preview

pikchr-preview is a VS Code extension that allows users to preview Pikchr
diagrams directly within the editor. It leverages the
[pikchr-cmd](https://github.com/zenomt/pikchr-cmd) tool to render Pikchr
diagrams as images in real-time as users edit their Pikchr code. The extension
supports previewing `.pikchr` files as well as Pikchr diagrams embedded within
Markdown files, utilizing VS Code's Markdown preview API.

## Features

* Real-time preview of Pikchr diagrams as you edit your `.pikchr` files.
* Support for rendering Pikchr diagrams embedded within Markdown files.
* Seamless integration with VS Code's Markdown preview API.

## Installation

To install the extension, search for `pikchr-preview` in the VS Code
Extensions Marketplace and click "Install". Once installed, you can open any
`.pikchr` file or a Markdown file containing Pikchr diagrams to see the
real-time preview in the editor.

## Usage

1. Open a `.pikchr` file or a Markdown file containing Pikchr diagrams.
2. The extension will automatically render the Pikchr diagrams in a preview pane.
3. As you edit the Pikchr code, the preview will update in real-time to reflect your changes.
